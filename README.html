README - AZX Token

Descrição

O AZX Token é um token BEP-20 que visa trazer inovação no mercado de ativos digitais, oferecendo aos seus holders benefícios como segurança, valorização e liquidez automática.

Características Principais

  • Token BEP-20 na Binance Smart Chain.
  • Taxa de 6,5% dividida entre reflexão, liquidez, marketing e queima.
  • Liquidez automática adicionada na PancakeSwap para estabilidade de preço.
  • Design inspirado no ouro, transmitindo valor e solidez.

Como Comprar

Você pode adquirir o AZX Token na PancakeSwap ou outras exchanges descentralizadas que listarem o token.
